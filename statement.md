# Problem Statement
Teachers need a quick way to run short MCQ quizzes and record student scores. Manual paper quizzes are time-consuming to grade. This system allows teachers to run quizzes and save results.

# Scope
- Console-based quiz; no network or DB required.
- Questions stored in CSV file.

# Target Users
Teachers and students for quick practice/assessment.

# High-Level Features
- Load questions from CSV
- Run shuffled quizzes
- Save results
